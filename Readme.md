Руководство пользователя:

1. Сервер

1.1 Параметры командной строки

-ip 	[address] - установка ip адреса сервера БД, где [address] IPv4 адрес
-port 	[port]    - установка порта для БД, где [port] целое число

1.2 Поддерживаемые команды

create 	[file]    - создание словаря частых слов, где [file] путь к файлу
update 	[file]    - обновление словаря частых слов, где [file] путь к файлу
clear		  - отчистка словаря частых слов

2. Клиент

2.1 Параметры командной строки

-ip 	[address] - установка ip адреса сервера, где [address] IPv4 адрес
-port 	[port]    - установка порта, где [port] целое число

2.2 Поддерживаемые команды

get 	[prefix]  - получение списка слов, начинающихся с указанного префикса, где [prefix] - строка
